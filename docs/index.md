# Totis - Gestión de bienes

TotisGdB es un sistema web orientado a la gestión y control operativo de activos fijos contables. Su propósito es centralizar información, facilitar solicitudes internas, mantener trazabilidad de movimientos y apoyar la generación de documentos relacionados con asignaciones, bajas, devoluciones y traspasos.

Esta documentación pública describe cómo está organizado el sistema, qué tecnologías utiliza y qué procesos cubre, sin publicar código fuente ni información sensible.

## Objetivos del sistema

- Centralizar la administración de activos fijos contables.
- Gestionar usuarios y perfiles operativos.
- Facilitar solicitudes de asignación, traspaso, baja y devolución.
- Registrar historial de movimientos para consulta y auditoría interna.
- Generar reportes y documentos de respaldo para procesos administrativos.
- Mantener flujos de autorización con controles de acceso.

## Alcance de esta documentación

La información publicada se mantiene en un nivel técnico descriptivo. Se explican componentes, responsabilidades, dependencias y flujos generales, pero no se incluyen implementaciones, archivos privados, credenciales, datos reales ni plantillas internas.

## Audiencia

Esta documentación está pensada para revisores técnicos, responsables de mantenimiento, usuarios interesados en la arquitectura del sistema y personas que quieran entender el propósito funcional de TotisGdB sin acceder al repositorio privado.
