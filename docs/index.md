# TotisGdB

TotisGdB es un sistema web orientado a la gestion y control operativo de activos fijos contables. Su proposito es centralizar informacion, facilitar solicitudes internas, mantener trazabilidad de movimientos y apoyar la generacion de documentos relacionados con asignaciones, bajas, devoluciones y traspasos.

Esta documentacion publica describe como esta organizado el sistema, que tecnologias utiliza y que procesos cubre, sin publicar codigo fuente ni informacion sensible.

## Objetivos del sistema

- Centralizar la administracion de activos fijos contables.
- Gestionar usuarios y perfiles operativos.
- Facilitar solicitudes de asignacion, traspaso, baja y devolucion.
- Registrar historial de movimientos para consulta y auditoria interna.
- Generar reportes y documentos de respaldo para procesos administrativos.
- Mantener flujos de autorizacion con controles de acceso.

## Alcance de esta documentacion

La informacion publicada se mantiene en un nivel tecnico descriptivo. Se explican componentes, responsabilidades, dependencias y flujos generales, pero no se incluyen implementaciones, archivos privados, credenciales, datos reales ni plantillas internas.

## Audiencia

Esta documentacion esta pensada para revisores tecnicos, responsables de mantenimiento, usuarios interesados en la arquitectura del sistema y personas que quieran entender el proposito funcional de TotisGdB sin acceder al repositorio privado.
